# 📈 Apex Finance Agent

> **AI-Powered Investment Analyst for Smarter Financial Decisions**

Apex Finance Agent is an intelligent AI-powered investment analysis platform that helps investors make informed financial decisions using real-time market data, Generative AI, and machine learning. It analyzes market trends, assesses portfolio risk, performs sentiment analysis, and provides personalized investment recommendations.

---

## 🚀 Features

- 📊 Real-Time Stock Market Analysis
- 🤖 AI-Powered Investment Recommendations
- 📈 Portfolio Optimization
- ⚠️ Risk Assessment & Management
- 📰 Financial News Analysis
- 😊 AI-Based Market Sentiment Analysis
- 💬 Conversational AI Investment Assistant
- 🔔 Smart Alerts & Notifications

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/Apex-Finance-Agent.git
cd Apex-Finance-Agent
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Start Backend

```bash
uvicorn main:app --reload
```

### Start Frontend

```bash
npm install
npm run dev
```
